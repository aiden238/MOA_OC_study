LLM Wiki 아키텍처 설계

LLM Wiki는 LLM 관련 지식을 구조화된 위키 형태로 관리하고, RAG를 통해 AI 에이전트가 필요할 때 지식을 검색하는 지식 관리 시스템입니다.

LLM Wiki의 핵심 개념:
기존 RAG는 단순히 텍스트 청크를 검색하지만, LLM Wiki는 지식을 체계적으로 구조화하여 더 정확하고 깊이 있는 검색을 가능하게 합니다.

Wiki+RAG 통합 전략:
1. 메타데이터 인덱싱: 태그, 카테고리, 관련 문서를 인덱스에 저장
2. 계층적 검색: 카테고리 필터 → 관련성 검색 → 연결 문서 확장
3. 쿼리 확장: 검색어의 동의어, 관련 개념을 자동으로 포함
4. 지식 그래프 탐색: 연결된 문서를 따라 관련 지식 수집

업데이트 관리:
- 새 연구/기법 등장 시 자동 감지 및 문서 추가
- 기존 문서 버전 관리 (변경 이력 유지)
- 커뮤니티 기반 검증 시스템

모델별 지식:
각 LLM 모델에 특화된 프롬프트 패턴, 제약사항, 최적 설정을 별도 섹션으로 관리:
- GPT-5 계열: reasoning_effort 설정, JSON 모드
- Gemini Flash: 빠른 응답, 멀티모달 지원
- Cerebras Qwen3: 온도 지원, 긴 컨텍스트 처리
